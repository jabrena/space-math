# Instalación de Python

## OSX

- https://brew.sh/
- https://docs.python-guide.org/starting/install3/osx/

```
brew install python
```

## Windows

- https://chocolatey.org/install
- https://community.chocolatey.org/packages/python/3.10.0

```
choco install python
```

## Test de instalación

```
python --version
```
